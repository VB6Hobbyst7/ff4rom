======================================================================================
NPC Placement								Phoenix Hacks
									12/17/2009
(based on Yousei's FF4 data)
======================================================================================

NPC Placement data is from 98500 to 998FF. Each item is 4 bytes.
To find the data for a particular map, reverse the pointer
bytes (located from 98200 to 984FF) and add them to 98500.
Each set of NPCs (for each map) is terminated by 0x00.

======================================================================================
Byte 0: NPC
Byte 1:
	bits 0-4:	Starting X position
	bit 5:		? (permanently deactivates NPC)
	bit 6:		?
	bit 7:		The NPC walks
Byte 2:
	bits 0-4:	Starting Y position
	bit 5:		? (permanently deactivates NPC)
	bits 6/7:	Characters can walk through the NPC
Byte 3:
	bits 0-1:	Direction the NPC faces
		00: up
		01: right
		10: down
		11: left
	bits 2-3:	Palette
	bit 4:		NPC turns when spoken to
	bit 5:		NPC moves in place even if not walking
	bits 6-7:	Movement speed
======================================================================================
